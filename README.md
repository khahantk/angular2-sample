# How to start
``` bash
git clone https://github.com/khahantk/angular2-sample.git angular2-sample
cd angular2-sample

#install the project's independencies
npm install
```
To run the sample application
``` bash
npm start
```
and then browse to [`http://localhost:3000`](http://localhost:3000).
